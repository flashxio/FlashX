fm.inner.prodR Documentation

Matrix inner product

Description

Inner product of a dense matrix with a dense vector/matrix.

Usage

fm.inner.prod(fm, mat, Fun1, Fun2)

Arguments

fm

A FlashMatrix matrix

mat

A FlashMatrix dense matrix.

Value

a FlashMatrix vector if the second argument is a vector; a FlashMatrix matrix if the second argument is a matrix.
