fm.conv.R2FMR Documentation

Convert a regular R object to a FlashMatrix object.

Description

If the R object is a matrix, ‘byrow’ determines how data in the generated FlashMatrix object is organized in memory.

Usage

fm.conv.R2FM(obj, byrow = FALSE)

Arguments

obj

a regular R object

byrow

a logical value to determine the data layout of a FlashMatrix matrix.

Value

a FlashMatrix object.
